## Run Locally

Clone the project

```bash
  gh repo clone beyondnaxos/Kasa 
 -------------------------------------
  https://github.com/beyondnaxos/Kasa
```

Go to the project directory

```bash
  cd Kasa
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
